# QuickBudget
iOS application for personal finance tracking built with SwiftUI.

## Technologies
- SwiftUI
- MVVM architecture
- SwiftData, UserDefaults
- Git

## Project Purpose
This project serves as a portfolio application demonstrating iOS development practices, app architecture, and UI design.
It does not contain any production APIs or personal data.

## Features
- Category-based expense tracking
- Add / edit financial records
- SwiftUI state management
- daily notification for the user
- Fast and easy to use
